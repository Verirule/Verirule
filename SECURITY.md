# Security Policy

## Reporting a Vulnerability

If you believe you have found a security issue in Verirule, please report it privately.

- Email: verirule@outlook.com
- Please include:
  - A clear description
  - Steps to reproduce
  - Potential impact
  - Any relevant logs/screenshots (avoid including secrets)

We aim to acknowledge reports within 72 hours.

## Supported Versions

Only the latest version on the main branch is supported.
